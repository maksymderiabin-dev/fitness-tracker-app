# 🏋️‍♂️ Фітнес Трекер (Full-Stack Застосунок)

Це веб-застосунок для відстеження спортивних тренувань, розроблений в рамках практики. Дозволяє користувачам додавати, переглядати, видаляти тренування та слідкувати за загальною статистикою.

## 🛠 Використані технології
* **Backend:** Nest.js, TypeScript, class-validator (In-memory БД)
* **Frontend:** React, Vite, TypeScript, React Router, Axios, Lucide React (іконки)

## ⚙️ Функціональність
* **CRUD операції:** Створення, читання та видалення тренувань.
* **Маршрутизація:** 3 окремі сторінки (Список, Форма додавання, Статистика).
* **Валідація:** Перевірка введених даних на клієнті та сервері (мінімальна довжина назви, обов'язкові поля).
* **Статистика:** Автоматичний підрахунок загальної кількості тренувань та витраченого часу.
